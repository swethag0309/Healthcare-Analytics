# Healthcare-Analytics
#### Healthcare Analytics in SQL


This repository demonstrates advanced proficiency in SQL for relational database management and data analysis, with a focus on healthcare datasets. It includes comprehensive examples showcasing various SQL join techniques (Inner, Left, Right, and Full Outer Joins) for merging and comparing complex tables. Aggregate functions such as COUNT and SUM are applied extensively for summarizing and reporting grouped data. The repository highlights expertise with SQL window functions, including RANK and DENSE_RANK, to perform sophisticated ranking and partitioned aggregations. Practical usage of subqueries and conditional logic enables flexible data filtering and classification. Date and time functions are integrated to analyze temporal patterns and trends, supporting insights into appointment durations, medication periods, and event-driven analytics. Through detailed reporting and query design, the repository reflects hands-on experience handling real-world healthcare scenarios for effective data-driven decision making.
